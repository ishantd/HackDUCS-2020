Problem Statement:

Differently abled and mute people use gesture-based communication methods to convey their thoughts and the most common medium are hand gestures. An innovation to help deaf adn mute people to communicate with others is a relevant solution.

Our Solution:

We have created a web API soon to be turned into a personal assistant that can help us converse with people who are Differently abled with a lot of ease. This API will help the user by:

  • learn ASL or ISL standard of sign language,
  • convert text into sign language,
  • translate using live camera feed sign language gestures into text
  • for Differently abled people who didn't have the oppurtunity to learn how to read, an OCR system that converts the written text to speech and also to Sign Language Gestures

Text to Sign Language Converter

Use this API to translate any language of your choice to Sign Language. It is easy to use and uses Python.

Sign Language to Text Converter

This API takes video input of hand gestures as in Indian Standards of Sign Langauge and translates them into any langauge of user's choice. This model google's tensforlow functions and VGG classification methods.
